###  How to use?
Add project namespace to the top of the project. After;
**Example Code:**
```csharp
//Initialize GithubAPI for User.
GithubNET.Users.SelectUser("MrCylops");

//Get User ID
var UserID = GithubNET.Users.Id;

//Print User ID
Console.WriteLine("User ID -> " + UserID.ToString());
```
